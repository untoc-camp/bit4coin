1. 개요

2024-1 UNTOC CAMP의 Bit4Coin팀(이하 개발팀)이 개발한 코인 자동매매 프로그램(이하 서비스)입니다. 해당 서비스는 binance 거래소 기반으로 코인을 거래하며, 선물거래를 바탕으로 자동매매가 진행됩니다.

2. 주의사항

해당 서비스를 통해 얻은 이익이나 손해 등 모든 수익은 개발팀의 책임이 없으며, 모두 사용자의 책임입니다. 이에 따른 배상은 없을 예정입니다.

7. 기능설명

해당 서비스에는 총 6가지의 기능이 있습니다.

7-1. goto : 코인 거래시 필요한 다양한 페이지가 있습니다.

  • Binance : 코인 거래소인 binance 페이지로 이동합니다.
  • Trading view : 코인 전략을 볼 수 있는 trading view 페이지로 이동합니다.
  • Mamual : 해당 사용설명서 페이지로 이동합니다.
  • GitHub : 개발팀의 GitHub페이지로 이동합니다.

7-2 symbols : 유명 코인들의 현재가와 시가대비를 보여줍니다.

  • symbols : 코인의 이름
  • Current Price : 코인의 현재가
  • 시가대비 : 당일 오전 9시 기준으로 한 코인 가격 상승률

7-3 home : 자동매매를 진입하는 페이지입니다. 자세한 설명은 "8. 자동매매 방법"에서 설명하겠습니다.

7-4. history : 거래 내역을 보여줍니다.

- 이미지 추가 예정-
  • 진입 가격 : 자동매매 시 진입한 진입 가격
  • 수익률 : ((현재가격 - 진입가격) / 현재가격) * 100
  • 보유량 : 구매한 코인의 보유량
  • 평가손익 : 현재가격 - 진입가격
  • 평가금액 : 보유량 * 현재가격
  • 손절 금액 : 손해를 바탕으로 포지션을 종료할 가격
  • 익절 금액 : 이익을 바탕으로 포지션을 종료할 가격
  • 테두리 : 파란색 - short / 빨간색 - long

7-5. more : 더보기 페이지입니다.

  • 개인정보관리 : 사용자의 아이디, 비밀번호를 관리할 수 있습니다.
  • API 관리 : 사용자의 API를 관리할 수 있습니다.
  • 사용 메뉴얼 : 해당 사용설명서 페이지로 이동합니다.
  • 이용약관 : 이용약관을 볼 수 있습니다.
  • 설정 : 설정 페이지로 이동합니다.(추후 추가 예정)
  • 전략 설명 : 자동매매에 이용되는 전략을 설명한다.
  • 마켓 : 자동매매를 할 수 있는 코인을 보여준다.

8. 자동매매 방법

- 이미지 추가 -

8-1. home page로 이동한다.

8-2. symbol에 자동매매를 진행할 코인의 이름을 작성한다.

8-3. percent에 현재 자본의 몇 퍼센를 자동매매할지 정한다.(0~1사이의 값)

8-4. leverge를 설정한다.

8-5. 오른쪽의 descript를 참고하여 전략을 선택한다.

8-6. ENTER버튼을 클릭한다.

8-7. 전략의 detail한 부분을 다시 읽고 Yes를 클릭하면 자동매매가 진행된다.

8-8. 자동매매 중지 버튼을 클릭하면 자동매매가 중지된다.
